# Security Policy

## Supported

Canonical Cloudberry archive on [art.adamsimms.xyz/cloudberry/archive](https://art.adamsimms.xyz/cloudberry/archive/).

## Reporting

Use GitHub security advisories on this repo or [art.adamsimms.xyz](https://github.com/adamsimms/art.adamsimms.xyz).

## Notes

- Never commit secrets or private keys.
- Media is on R2; this repo is the static archive source assembled into art Pages.
- In scope: XSS, secret exposure, poisonous catalog/asset paths. Out of scope: third-party services (Cloudflare, Mapbox, Vimeo) unless this repo’s integration introduced the issue.
